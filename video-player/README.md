# Imitation YouTube Video Player
This app was created by following along with Udemy tutorial created by Stephen Grider.  https://www.udemy.com/react-redux/learn/v4/content  The app makes API calls to the YouTube API to search for videos; the app displays a list of matching videos for users to select and play.  This is the beginning app created in this tutorial and introduces React and ajax requests in React.

The app was created off the ReduxSimpleStarter and the original readme is below.

### Getting Started

There are two methods for getting started with this repo.

#### Familiar with Git?
Checkout this repo, install dependencies, then start the gulp process with the following:

```
> git clone https://github.com/StephenGrider/ReduxSimpleStarter.git
> cd ReduxSimpleStarter
> npm install
> npm start
```

#### Not Familiar with Git?
Click [here](https://github.com/StephenGrider/ReactStarter/releases) then download the .zip file.  Extract the contents of the zip file, then open your terminal, change to the project directory, and:

```
> npm install
> npm start
```
